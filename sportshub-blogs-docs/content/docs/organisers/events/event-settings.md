## Overview

Event settings allow you to configure how your event appears and behaves for participants. You can access event settings from the **Settings** tab when viewing an event (or recurring event template) in your [Organiser Hub](https://www.sportshub.net.au/organiser).

## Show Attendees on Event Page

This setting controls whether a **"Who's going"** section appears on your public event page, displaying the names of registered attendees to potential participants.

### What Participants See

When enabled, a **"Who's going"** section appears on the public event page. Each attendee is displayed as a card with their initial in a circular avatar alongside their full name.

- **Desktop:** The grid shows 6 attendees at a glance.
- **Mobile:** The grid shows 8 attendees (taking advantage of the narrower card layout).
- **View all:** If there are more attendees than can fit in the preview, a **"View all (N)"** link appears that opens a full scrollable list in a modal.

The design is clean, minimal, and fits right into the existing event page layout.

### Privacy

Only full names are displayed. No email addresses, phone numbers, ticket types, payment information, or any other private data is exposed. The feature is opt-in and disabled by default.

### How to Enable

#### For Single Events

1. Head to your **Organiser Hub** and open the event you want to update.
2. Navigate to the **Settings** tab.
3. Toggle **"Show Attendees on Event Page"** to on.

The attendee list will appear on the public event page immediately.

#### For Recurring Events

If you manage recurring events, you can enable this on the **recurring event template** as well. Every new event generated from that template will automatically show attendees.

1. Open your recurring event template in the Organiser Hub.
2. Go to the **Settings** tab.
3. Toggle **"Show Attendees on Event Page"** to on.

All future events created from that template will inherit the setting.

### When to Use This Setting

Consider enabling this when:

- You want to leverage social proof to encourage more bookings
- Your events attract a community where attendees may know each other
- You run regular events where participants like to see who else is coming

You can toggle the setting on or off at any time, and the change takes effect immediately.
