# Sportshub Event Analytics Chrome Extension

This is the folder containing the code for the sportshub chrome extension.

It uses webpack which is built to the `./dist` folder.

When loading the unpacked extension to chrome, you should first run `npm run build` in this folder to compile the webpack to the `./dist` folder and then reload the chrome extension on chrome. See here for more information: https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world
